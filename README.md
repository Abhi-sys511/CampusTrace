# CampusTrace

CampusTrace is a digital Lost and Found platform developed to help students and staff report, search, and recover lost belongings within a campus. 

It was published in the IRO Journal – TCSST 2024 conference.

## 🚀 Features

- Report lost and found items with details
- Admin login for item approval
- User-friendly dashboard
- Search and filter lost/found items
- Built-in contact feature for item claim

## 🛠️ Technologies Used

- Python
- Flask
- SQLite
- HTML5, CSS3
- Bootstrap

## 🏫 Paper Publication

Published in: *IRO Journals – Transactions on Computer Science & Smart Technology (TCSST), 2024*  
🔗 [View Paper](https://irojournals.com/tcsst/article/view/6/2/6)
